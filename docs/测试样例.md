## 1. 基础对话

### 1.1 新会话 — 简单问候

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"user_id": "tester", "message": "你好，请介绍一下你自己"}'
```

**预期响应:**
```json
{
  "session_id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
  "answer": "你好！我是一个通用 AI 助手，可以使用多种工具来帮你完成任务...",
  "error": null,
  "loop_count": 0
}
```

### 1.2 继续已有会话

```bash
# 把 {SESSION_ID} 替换为 1.1 返回的 session_id
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"session_id": "{SESSION_ID}", "user_id": "tester", "message": "刚才我们聊了什么？"}'
```

**预期响应:** `answer` 中应包含对上一轮对话的引用

## 2. 工具调用

### 2.1 文件读取

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"user_id": "tester", "message": "帮我读取 pyproject.toml 的内容"}'
```

**预期响应:** `answer` 中输出 pyproject.toml 内容，`loop_count` >= 1（说明走了一轮工具调用）

### 2.2 目录浏览

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"user_id": "tester", "message": "列出当前项目的所有 Python 文件"}'
```

**预期:** Agent 调用 `search_file` 工具，列出 `*.py` 文件

### 2.3 代码搜索

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"user_id": "tester", "message": "在 src 目录下搜索所有包含 \"async def\" 的文件和行"}'
```

**预期:** Agent 调用 `grep_content` 工具，返回匹配结果

### 2.4 文件写入

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"user_id": "tester", "message": "创建一个文件 /tmp/pyagent-test.txt，内容是: Hello from PyAgent!"}'
```

**预期:** Agent 调用 `write_file` 工具，确认写入成功

## 3. 权限控制

### 3.1 高危命令被拦截

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"user_id": "tester", "message": "用 shell 执行 rm -rf /tmp/test"}'
```

**预期响应:** Agent 应该拒绝执行 `exec_shell`（高风险工具），返回类似"权限拒绝"或"高风险操作"的提示

### 3.2 低风险命令正常放行

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"user_id": "tester", "message": "读取 README.md 文件内容"}'
```

**预期:** 正常执行（`read_file` 是 low risk）

## 5. 会话管理

### 5.1 查看会话列表

```bash
curl "http://127.0.0.1:8000/session/list?user_id=tester"
```

**预期:**
```json
[
  {
    "id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx",
    "user_id": "tester",
    "title": "你好，请介绍一下你自己",
    "status": "active",
    "message_count": 1,
    "created_at": "2025-05-26 ...",
    "updated_at": "2025-05-26 ..."
  }
]
```

### 5.2 查看单个会话详情

```bash
curl "http://127.0.0.1:8000/session/{SESSION_ID}"
```

**预期:** 返回该会话的完整元数据

### 5.3 关闭会话

```bash
curl -X POST "http://127.0.0.1:8000/session/{SESSION_ID}/close"
```

**预期:**
```json
{"status": "closed"}
```

### 5.4 验证关闭后无法发送消息

```bash
curl -X POST http://127.0.0.1:8000/chat \
  -H "Content-Type: application/json" \
  -d '{"session_id": "{SESSION_ID}", "user_id": "tester", "message": "还能说话吗？"}'
```

**预期:**
```json
{"detail": "Session is closed"}
```
状态码: 400

### 5.5 删除会话

```bash
curl -X DELETE "http://127.0.0.1:8000/session/{SESSION_ID}"
```

**预期:**
```json
{"deleted": true}
```

### 5.6 删除后查询返回 404

```bash
curl "http://127.0.0.1:8000/session/{SESSION_ID}"
```

**预期:**
```json
{"detail": "Session not found"}
```
状态码: 404

## 7. 批量测试脚本

保存为 `test_api.sh`，一次运行全部基础测试：

```bash
#!/bin/bash
BASE="http://127.0.0.1:8000"

echo "=== 1. 健康检查 ==="
curl -s "$BASE/health" | python -m json.tool

echo -e "\n=== 2. 新会话 ==="
SESSION=$(curl -s -X POST "$BASE/chat" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"tester","message":"你好"}')
echo "$SESSION" | python -m json.tool
SID=$(echo "$SESSION" | python -c "import sys,json; print(json.load(sys.stdin)['session_id'])")

echo -e "\n=== 3. 继续会话 ==="
curl -s -X POST "$BASE/chat" \
  -H "Content-Type: application/json" \
  -d "{\"session_id\":\"$SID\",\"user_id\":\"tester\",\"message\":\"刚才说了什么？\"}" | python -m json.tool

echo -e "\n=== 4. 工具调用：读取文件 ==="
curl -s -X POST "$BASE/chat" \
  -H "Content-Type: application/json" \
  -d '{"user_id":"tester","message":"读取 pyproject.toml 的前5行"}' | python -m json.tool

echo -e "\n=== 5. 会话详情 ==="
curl -s "$BASE/session/$SID" | python -m json.tool

echo -e "\n=== 6. 关闭会话 ==="
curl -s -X POST "$BASE/session/$SID/close" | python -m json.tool

echo -e "\n=== 7. 删除会话 ==="
curl -s -X DELETE "$BASE/session/$SID" | python -m json.tool

echo -e "\n=== 全部测试完成 ==="
```

## 8. Python 客户端示例

```python
"""PyAgent Python 客户端测试"""
import requests

BASE = "http://127.0.0.1:8000"


def test_chat():
    """测试对话功能"""
    # 新会话
    resp = requests.post(f"{BASE}/chat", json={
        "user_id": "python_tester",
        "message": "你好，用一句话介绍 Python",
    })
    data = resp.json()
    print(f"新会话: session_id={data['session_id']}")
    print(f"回答: {data['answer']}")
    print(f"循环次数: {data['loop_count']}")
    assert resp.status_code == 200
    assert data["session_id"] is not None
    assert data["answer"] is not None
    return data["session_id"]


def test_continue_session(session_id: str):
    """测试会话延续"""
    resp = requests.post(f"{BASE}/chat", json={
        "session_id": session_id,
        "user_id": "python_tester",
        "message": "你刚才说的语言是什么版本？",
    })
    data = resp.json()
    print(f"继续会话回答: {data['answer']}")
    assert resp.status_code == 200


def test_tool_read_file():
    """测试文件读取工具"""
    resp = requests.post(f"{BASE}/chat", json={
        "user_id": "python_tester",
        "message": "用 read_file 读取 pyproject.toml",
    })
    data = resp.json()
    print(f"工具调用回答: {data['answer'][:200]}...")
    assert resp.status_code == 200
    assert data["loop_count"] >= 1  # 至少一轮工具调用


def test_session_crud(session_id: str):
    """测试会话 CRUD"""
    # 列表
    resp = requests.get(f"{BASE}/session/list", params={"user_id": "python_tester"})
    assert resp.status_code == 200

    # 详情
    resp = requests.get(f"{BASE}/session/{session_id}")
    assert resp.status_code == 200
    assert resp.json()["id"] == session_id

    # 关闭
    resp = requests.post(f"{BASE}/session/{session_id}/close")
    assert resp.json()["status"] == "closed"

    # 关闭后发送消息应失败
    resp = requests.post(f"{BASE}/chat", json={
        "session_id": session_id,
        "user_id": "python_tester",
        "message": "还能说话吗？",
    })
    assert resp.status_code == 400

    # 删除
    resp = requests.delete(f"{BASE}/session/{session_id}")
    assert resp.json()["deleted"] is True

    # 删除后查询 404
    resp = requests.get(f"{BASE}/session/{session_id}")
    assert resp.status_code == 404


def test_health():
    """测试健康检查"""
    resp = requests.get(f"{BASE}/health")
    assert resp.json()["status"] == "ok"


def test_memory():
    """测试记忆归档与召回"""
    # 保存偏好
    resp = requests.post(f"{BASE}/chat", json={
        "user_id": "python_tester",
        "message": "记住，我偏好用 Poetry 管理 Python 依赖",
    })
    assert resp.status_code == 200

    # 查询偏好
    resp = requests.post(f"{BASE}/chat", json={
        "user_id": "python_tester",
        "message": "我喜欢的 Python 包管理器是什么？"
    })
    print(f"记忆召回回答: {resp.json()['answer']}")


if __name__ == "__main__":
    print("开始 PyAgent 测试...")
    test_health()
    print("✓ 健康检查通过")

    sid = test_chat()
    print("✓ 新会话通过")

    test_continue_session(sid)
    print("✓ 会话延续通过")

    sid2 = test_chat()
    test_tool_read_file()
    print("✓ 工具调用通过")

    test_memory()
    print("✓ 记忆测试通过")

    test_session_crud(sid2)
    print("✓ 会话 CRUD 通过")

    print("\n全部测试通过！")
```
